## 1. 서비스 개요

### 1.1 서비스 한 줄 정의
**"누구나 5분 안에 전문 리서치를 시작할 수 있는 온라인 설문 플랫폼"**

### 1.2 배경 및 목적
- 기업·스타트업의 시장조사 수요는 매년 증가하지만, 전통 리서치 방식은 비용(수백만 원~)과 기간(2주+)이 과도하게 발생
- 오픈서베이 등 국내 선발 플랫폼이 B2B 고가 요금제 중심으로 이동하면서 **중소기업·1인 창업자·대학(원)생** 세그먼트에 공백 발생
- Tommy Survey는 **셀프서브(self-serve) UI + 검증된 패널 + 실시간 분석** 조합으로 해당 공백을 공략

### 1.3 핵심 가치 제안 (Value Proposition)
| 대상 | 기존 불편 | Tommy Survey 해결책 |
|---|---|---|
| 스타트업 PM | 리서치 에이전시 비용 부담 | 설문 빌더 + 패널 풀로 직접 운영 |
| 마케터 | 엑셀 수기 집계 | 실시간 대시보드 & 자동 크로스탭 |
| 대학(원)생 | 지인 섭외 한계 | 타깃 패널 필터링 + 소액 결제 |
| 패널(응답자) | 보상 불투명 | 포인트 즉시 적립 + 현금 전환 |

## 2. 타깃 사용자 분석

### 2.1 사용자 유형 (2 Track)

#### Track A — 의뢰자 (Survey Creator)
- **주요 페르소나**
  - 스타트업 PM/CPO: 제품 PMF 검증, UX 리서치
  - 중소기업 마케터: 브랜드 인지도·광고 효과 측정
  - 대학원생: 논문 설문, IRB 데이터 수집
  - 프리랜서 리서처: 클라이언트 대행 납품
- **핵심 니즈**: 빠른 설문 제작 / 신뢰도 높은 표본 / 보기 쉬운 분석 리포트
- **결제 민감도**: 중간 (건당 10~50만 원 수용, SaaS 월정액 거부감 낮음)

#### Track B — 패널 (Survey Respondent)
- **주요 페르소나**
  - 20~40대 직장인: 퇴근 후 틈새 수입
  - 대학생: 소액 용돈
  - 주부: 재택 부업
- **핵심 니즈**: 설문 간편 참여 / 보상 즉시 지급 / 개인정보 신뢰
- **이탈 원인**: 설문 길이 과다 / 보상 낮음 / 탈락률 높음

### 2.2 사용자 여정 (High-level)
```
[의뢰자]
회원가입 → 설문 빌더 → 타깃 설정 → 결제 → 배포 → 실시간 모니터링 → 리포트 다운로드

[패널]
앱/웹 회원가입 → 프로필 등록 → 맞춤 설문 수신 → 응답 → 포인트 적립 → 출금
```

## 4. 핵심 기능 정의

### 4.1 설문 빌더 (Survey Builder)

| 기능 | 상세 설명 | 우선순위 |
|---|---|---|
| 문항 유형 | 단일/복수 선택, 척도(리커트), 순위, 빈칸 입력, 행렬형, NPS | P0 |
| 드래그앤드롭 | 문항 순서 자유 변경 | P0 |
| 로직 분기 | 응답에 따라 특정 문항 건너뛰기/이동 | P0 |
| 템플릿 라이브러리 | 고객만족(CSAT), 브랜드 인지도, UX 리서치 등 30+ 템플릿 | P1 |
| 미리보기 | 모바일/PC 실시간 미리보기 | P0 |
| 진행률 표시 | 응답자 화면에 진행 바 노출 | P1 |
| 설문 만료 설정 | 날짜·응답 수 기준 자동 마감 | P1 |
| 다국어 지원 | 한국어·영어 (초기) | P2 |

### 4.2 타깃 패널 설정

| 필터 항목 | 옵션 예시 |
|---|---|
| 인구통계 | 성별, 연령대, 지역, 학력, 직업, 결혼 여부 |
| 소비 행동 | 구매 카테고리, 월 소비 수준 |
| 미디어 이용 | 주 이용 SNS, OTT 구독 여부 |
| 특수 패널 | 임산부, 반려동물 보호자, 운전자 등 |

- 실시간 **표본 크기 예측** (필터 변경 시 즉시 갱신)
- 최소 표본 50명 ~ 최대 10,000명
- **스크리너 문항** 추가로 응답 자격 사전 검증

### 4.3 실시간 리포트 & 분석

- **요약 카드**: 응답률, 완료 수, 평균 소요 시간, 중도 이탈 문항
- **문항별 시각화**: 막대/파이 차트, 워드클라우드 (주관식)
- **크로스탭**: 2개 문항 교차 분석, 유의미한 차이 자동 하이라이트
- **필터**: 특정 응답자 세그먼트만 추출해 재분석
- **다운로드**: PDF 리포트, Excel 원본, SPSS(.sav) 포맷

### 4.4 패널 참여 경험

- 설문 평균 길이: **5~10분 이내 권장** (빌더에서 예상 시간 자동 계산)
- 보상: **건당 100~500 포인트** (1pt = 1원 환산 기준)
- 포인트 출금: 5,000pt 이상 현금 전환 (계좌 이체, 카카오페이, 네이버페이)
- 탈락(스크리너 통과 실패) 시 **소액 위로 포인트** 지급 → 이탈률 감소
- 프로필 완성도 높을수록 맞춤 설문 우선 배정

### 4.5 알림 시스템

| 수신자 | 알림 조건 | 채널 |
|---|---|---|
| 의뢰자 | 목표 응답 수 달성, 설문 마감 임박 | 이메일, 앱 푸시 |
| 의뢰자 | 크레딧 잔액 부족 | 이메일, SMS |
| 패널 | 맞춤 설문 신규 등록 | 앱 푸시, 카카오 알림톡 |
| 패널 | 포인트 출금 완료 | 앱 푸시, SMS |

## 5. 주요 화면 흐름 (User Flow)

### 5.1 의뢰자 — 설문 생성 ~ 결과 수령
```
[설문 만들기] 클릭
    │
    ▼
빌더 화면 진입
    ├── 템플릿 선택 or 빈 설문
    ├── 문항 추가·편집
    └── 미리보기 확인
    │
    ▼
[다음: 타깃 설정]
    ├── 인구통계 필터 설정
    ├── 표본 수 입력
    └── 예상 완료 기간 표시
    │
    ▼
[다음: 결제]
    ├── 크레딧 사용 or 카드 결제
    └── 최종 금액 확인
    │
    ▼
[배포 시작] → 패널에 설문 발송
    │
    ▼
실시간 대시보드 (응답 누적 확인)
    │
    ▼
[마감] → 최종 리포트 생성 → 다운로드
```

### 5.2 패널 — 설문 참여 흐름
```
앱 홈 피드에서 설문 확인
    │
    ▼
설문 소개 화면 (예상 시간, 보상 확인)
    │
    ▼
스크리너 문항 응답
    ├── 통과 → 본 설문 진행
    └── 탈락 → 위로 포인트 지급 후 종료
    │
    ▼
본 설문 응답 (로직 분기 포함)
    │
    ▼
완료 화면 → 포인트 즉시 적립 확인
```

## 6. 요금제 설계

### 6.1 의뢰자 요금제

| 플랜 | 월 금액 | 포함 내용 |
|---|---|---|
| **Free** | 0원 | 자체 링크 배포, 응답 100건, 기본 문항 10개 |
| **Starter** | 29,000원 | 패널 배포 300응답/월, 전체 문항 유형, 기본 리포트 |
| **Business** | 99,000원 | 패널 배포 1,000응답/월, 크로스탭, 데이터 다운로드 |
| **Enterprise** | 별도 문의 | 무제한 응답, 전담 PM, 커스텀 패널, SLA 보장 |

> 패널 배포는 **응답 건당 과금** 방식 병행 (크레딧 충전 후 차감)

### 6.2 패널 보상 구조
- 기본 보상: 5분 기준 200pt (분당 40pt)
- 설문 길이·난이도에 따라 의뢰자가 보상 레벨 선택 (100/200/500pt)
- 의뢰자 지불 금액의 **60%** → 패널 보상, **40%** → 플랫폼 수수료

## 7. 기술 스택 (권장)

| 영역 | 기술 | 선택 이유 |
|---|---|---|
| Frontend | React + TypeScript | 컴포넌트 재사용, 타입 안정성 |
| 설문 빌더 | react-beautiful-dnd | 드래그앤드롭 UX |
| 차트 | Recharts / Chart.js | 경량, 커스터마이즈 용이 |
| Backend | Node.js (Express) + Firebase Functions | 서버리스, 빠른 초기 개발 |
| DB | Firestore (NoSQL) | 실시간 응답 집계에 적합 |
| 인증 | Firebase Auth (소셜 로그인) | 패널·의뢰자 멀티 역할 지원 |
| 결제 | 토스페이먼츠 or 아임포트 | 국내 카드, 간편결제 지원 |
| 푸시 알림 | Firebase Cloud Messaging (FCM) | 웹+앱 통합 |
| 호스팅 | Firebase Hosting + CDN | 글로벌 빠른 응답 |
| 분석 | Google Analytics 4 + BigQuery | 행동 로그 심화 분석 |

## 8. 비기능 요구사항

### 8.1 성능
- 설문 응답 페이지 **LCP < 1.5초** (느린 모바일 기준)
- 실시간 리포트 업데이트 **지연 < 3초**
- 동시 응답자 **10,000명** 처리 가능 (수평 확장 설계)

### 8.2 보안 · 개인정보
- 개인정보보호법 준수: 응답 데이터 익명화 처리
- HTTPS 필수, Firebase Security Rules로 데이터 접근 제어
- 패널 연령 인증 (만 19세 미만 제한 설문 별도 처리)
- GDPR 대응 구조 (유럽 진출 대비)

### 8.3 접근성
- WCAG 2.1 AA 기준 충족
- 스크린리더 호환 (ARIA 레이블)
- 색약 사용자 고려 색상 팔레트
